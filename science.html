R Data Science

Introduction

Ever since I was tasked to analyze multi-year All Payers All Claims dataset with (very) limited computational resources I had at the time, I started using 'data.table' and 'sqldf' as my primary mode of processing data. 'data.table' is much more efficient at processing data compared to similar code written in base R. 'sqldf' is very useful to manage dataset using popular sql syntax. Another advantage of using sqldf is that if your organization has a data warehouse, it is possible to access it with your credential directly in R hence making it easier to streamline your workflow.

I will be posting some examples from my own work as well as link to some of the materials that I find useful when working in R.
